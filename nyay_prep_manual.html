Chapter 1
Getting Started
शुरुआत करें · सुरुवात करा
What you need before starting
  • 1
    Python 3.9 or higher — Download from python.org if not installed. Check by running python --version in terminal.
    On Mac, use python3 --version instead.
  • 2
    VS Code — Download from code.visualstudio.com. Install the Python extension from the Extensions marketplace.
  • 3
    Anthropic API Key — Sign up free at console.anthropic.com. You get $5 free credits on signup — enough for hundreds of queries.
  • 4
    Nyay Prep project files — Unzip the nyay_prep_v3.zip folder to any location on your computer (e.g. Desktop or Documents).
Installation steps
  • 1
    Open the project in VS Code
    File → Open Folder → select the nyay_prep_v3 folder
  • 2
    Open the terminal
    Press Ctrl + ` (backtick) on Windows, or Cmd + ` on Mac. The terminal opens at the bottom of VS Code.
  • 3
    Create a virtual environment
    Windows: python -m venv venv then venv\Scripts\activate
    Mac/Linux: python3 -m venv venv then source venv/bin/activate
    You will see (venv) appear at the start of your terminal line when active.
  • 4
    Install required packages
    pip install -r requirements.txt
    This takes 2–3 minutes. You will see packages downloading.
  • 5
    Create your .env file
    In VS Code, create a new file named .env in the nyay_prep_v3 folder. Add this line:
    ANTHROPIC_API_KEY=sk-ant-your-key-here
    Replace sk-ant-your-key-here with your actual key from console.anthropic.com
  • 6
    Run the app
    streamlit run app.py
    Your browser opens automatically at http://localhost:8501
Success! You will see the Nyay Prep app with a dark navy sidebar and the Indian tricolour strip at the top.

Chapter 3
Switching the UI Language
भाषा बदलें · भाषा बदला

Nyay Prep supports three interface languages. Switching changes all labels, buttons, placeholders, and messages throughout the entire app.

Language switcher — located at the top of the sidebar
💡 Note — Output Language vs UI Language
The UI Language (EN / हिंदी / मराठी) controls the app interface — labels, buttons, menus.
The Output Language dropdown inside each tool controls what language Claude drafts/responds in.
You can have the UI in Marathi but ask Claude to draft a document in English — they are independent.
How to switch languages
  • 1
    Look at the top of the sidebar — you will see three buttons: EN | हिंदी | मराठी
  • 2
    Click your preferred language button — the active one is highlighted in saffron orange with a ✓ checkmark
  • 3
    The entire app rerenders in that language — sidebar menu, page titles, all field labels, buttons, placeholders, and messages switch immediately
Chapter 4 — Tool 1
Document Drafting
दस्तावेज़ मसौदा · दस्तावेज मसुदा

Draft any Indian court document — plaints, petitions, affidavits, legal notices, contracts — in English, Hindi, or Marathi. Claude follows Indian legal format and includes all mandatory sections.

Document Drafting — step by step
  • 1
    Select Document Category — Choose from Civil (CPC), Criminal (CrPC), Constitutional/Writ, or Other.
    The second dropdown will auto-populate with relevant document types for your chosen category.
  • 2
    Select Document Type — e.g. Plaint, Bail Application, Writ Petition, Affidavit, NDA, etc.
  • 3
    Select Court Category then Court — e.g. High Courts → Bombay High Court. The court name is passed to Claude to apply the correct jurisdiction rules.
  • 4
    Enter Parties — Name both sides e.g. "ABC Pvt Ltd (Petitioner) vs State of Maharashtra (Respondent)".
  • 5
    Describe the Facts — Write the key facts in plain language. No need for legal language — Claude will apply the right legal format. Then click Draft document.
💡 Tips for better drafts
Include dates, section numbers, and amounts in your facts. e.g. "Notice dated 15/03/2025 under Section 148 for AY 2022-23 demanding ₹45L" will produce a more precise draft than vague facts.

Chapter 6 — Tool 3
Contract Review
अनुबंध समीक्षा · करार पुनरावलोकन

Upload or paste any contract. Claude identifies risk clauses, missing clauses, obligations of each party, and suggests specific redline changes — under Indian Contract Act 1872.

  • 1
    Select contract type — Service Agreement, NDA, Employment, Lease, Sale, etc.
  • 2
    Select your client's role — e.g. Service Provider, Tenant, Buyer. This tells Claude which party's interests to protect.
  • 3
    Select governing law — Indian Law, Maharashtra, Delhi, etc.
  • 4
    Paste or upload the contract — Use the Paste tab for copy-paste, or Upload tab for PDF/DOCX files.
  • 5
    Click Review contract — Results show risk level (🔴 High / 🟡 Medium / 🟢 Low), clause-by-clause analysis, and suggested rewrites.
⚠️ Important
The redline suggestions show original vs suggested text side by side. Always have the advocate review before sending to opposite party.

Chapter 16
Troubleshooting & FAQ
समस्या समाधान
Common errors and fixes
❌ ModuleNotFoundError: No module named 'anthropic'
Your virtual environment is not active, or packages were not installed.

Fix: Run source venv/bin/activate (Mac) or venv\Scripts\activate (Windows), then pip install -r requirements.txt
❌ ANTHROPIC_API_KEY not found
The .env file is missing or the key is wrong.

Fix: Create a file named .env in the nyay_prep_v3 folder. Add: ANTHROPIC_API_KEY=sk-ant-your-key-here
❌ PDF extraction failed
PyMuPDF may not be installed correctly.

Fix: Run pip install PyMuPDF --upgrade
❌ Streamlit shows "Error: could not parse JSON"
Claude returned an unexpected response format.

Fix: Click the run button again — this is usually a one-time API hiccup. If it persists, simplify your input text.
✅ App runs slowly on first query? This is normal — Claude API takes 5–10 seconds per query. Subsequent queries in the same session may be faster.
Frequently asked questions
  • Q
    Can I use Hindi/Marathi text directly in the facts or question fields?
    Yes. All text input boxes accept English, Hindi, Marathi, or mixed text. The language is auto-detected and Claude analyses natively.
  • Q
    Does the tool save my documents?
    No. Nyay Prep does not store any documents. All data stays in your browser session and is cleared when you close the app. Use the Download buttons to save your output.
  • Q
    What is the document size limit?
    Documents are truncated to ~7,000 characters before being sent to Claude. For very long documents (100+ pages), paste the most relevant sections.
  • Q
    How much does it cost to run?
    With the default Claude Sonnet model, each query costs approximately ₹1–4. Switching to Claude Haiku in core/claude_client.py reduces this to under ₹0.50 per query.
  • Q
    Can I share this with my team?
    Yes. Deploy on Streamlit Cloud (free) and share the URL. All team members can use it from any browser — no installation needed. See README.md for deployment instructions.
  • Q
    Is the output legally accurate?
    Claude is a highly capable AI but is not a substitute for professional legal advice. Always review AI-drafted documents before filing. Nyay Prep is a drafting and research aid — the advocate is responsible for the final document.
